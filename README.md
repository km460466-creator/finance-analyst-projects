# Finance Analyst Projects (FP&A Focus)

This repository contains senior-level Excel-based finance analysis projects
built to solve real-world corporate finance problems.

## Projects

### 1. Receivables & Cash Leakage Analysis
- Customer-wise ageing and overdue analysis
- Working capital and liquidity assessment
- Identification of recoverable cash without borrowing

### 2. Cost Inflation vs Inefficiency Analysis
- Unit cost comparison across periods
- Separation of inflationary impact vs internal inefficiencies
- Margin protection and cost control insights

## Tools Used
- Microsoft Excel
- Financial Analysis & FP&A Techniques

## Note
All data used in these projects is simulated but based on realistic
business scenarios derived from professional experience.
